# **App Name**: Winter Warfare

## Core Features:

- Tower Placement: Allow players to drag and drop towers onto a grid-based battlefield.
- Tower Defense Gameplay: Implement wave-based enemy spawning and tower-based defense mechanisms.
- Currency and Economy: Earn currency by defeating enemies; spend it to buy and upgrade towers.
- Arsenal Suggestor: Suggest the towers most suited for the incoming enemy wave. The AI tool will determine the best combination based on known strengths and weaknesses of enemies.

## Style Guidelines:

- Primary color: Use a vibrant red (#D32F2F) to capture the Christmas spirit and military intensity, suggesting both festivity and high-stakes action. 
- Background color: A desaturated, dark, bluish-gray (#263238) to create a contrast that enhances the visibility of towers and enemies, resembling a crisp, snowy, overcast sky.
- Accent color: A warm, muted orange (#FF8F00) to highlight interactive elements like buttons and selection indicators.
- Body and headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Code font: 'Source Code Pro', a monospace font, can be used to show game debug messages.
- Use clean, modern icons for the tower menu and control panel.
- Subtle animations for tower firing and enemy movement to add visual interest.
